# Windows-клиент

В этой папке лежит клиентская часть схемы. Серверная часть остаётся в корне репозитория и не смешивается с кодом клиента.

Текущий этап:

- tray-приложение для Windows;
- язык `C++`;
- графический стек `Win32 API`;
- сборка через `CMake`.

Что сейчас закрывается этим подпроектом:

- иконка в трее;
- открытие главного окна по левому клику;
- контекстное меню по правому клику;
- фоновая работа после закрытия окна;
- режим запуска без показа главного окна;
- защита от повторного запуска через именованный mutex;
- сборка на GitHub Actions с публикацией `exe`.

Дальше сюда будут добавляться клиентские модули из схемы:

- модуль авторизации и лицензий;
- модуль обновления баз;
- локальное хранилище сигнатур;
- интеграция со сканирующим модулем.
